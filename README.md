# 오늘의 루틴 생성기 (Today's Routine Generator)

날씨와 감정 상태를 기반으로 맞춤형 일상 루틴을 제안하는 웹 서비스입니다.

## 주요 기능
- 실시간 날씨 정보 연동
- 감정 상태 기반 루틴 추천
- 반응형 디자인
- 다크모드 지원
- 모던한 UI/UX

## 기술 스택
- HTML5/CSS3
- JavaScript (ES6+)
- 날씨 API 연동
- 반응형 웹 디자인

## 구매 시 제공 사항
- 전체 소스코드
- 설치 가이드
- 6개월 기술 지원
- API 연동 가이드
